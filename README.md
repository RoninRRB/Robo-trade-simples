# Robo-trade-simples
Robô simples que busca preço e da alerta de compra/venda de BTC e ETH.

Projeto simples usando Streamlit e Binance API
## Funcionalidades
- Monitoramento de preço em tempo real
- Sinais de compra e venda
- Interface web com Streamlit
